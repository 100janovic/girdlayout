# GridLayout.js

Small script for Grid Layout.

# Usage

```

var grid = new GridLayout('promos-wrapper', { // id
   selector: '.promo', 
   speed: 400, 
   colwidth: 200, 
   debuglog: false, 
   animate: true 
});

```
